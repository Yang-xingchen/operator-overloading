# operator-overloading
实现Java版本的操作符重载功能。为方便使用诸如`BigDecimal`类库。

> [!IMPORTANT]
> 目前还在早期demo阶段，请勿使用于生产环境，欢迎提交issues及PR

使用示例: [Main.java](example/src/main/java/org/yangxc/example/Main.java)
